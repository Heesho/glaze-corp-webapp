## Network Information

### Base Mainnet

```
Network Name: Base
Chain ID: 8453
Currency: ETH
RPC URL: https://mainnet.base.org
Block Explorer: https://basescan.org
```

### Adding Base to MetaMask

1. Open MetaMask
2. Click network dropdown
3. Click "Add Network"
4. Enter the above details
5. Click "Save"

Or visit [chainlist.org](https://chainlist.org) and search for Base.

## Getting Tokens

### ETH on Base

Options to get ETH on Base:
1. **Bridge from Ethereum**: Use the [Base Bridge](https://bridge.base.org)
2. **Buy directly**: Coinbase supports direct Base deposits
3. **Cross-chain swap**: Use bridges like Stargate, Hop, or Across

### DONUT

1. **Mine it**: Become the active miner
2. **Buy on Uniswap**: Swap ETH for DONUT
3. **Earn through governance**: Claim DONUT from strategy bribes

## Community Guidelines

### Discord Etiquette

- Be respectful to all members
- No spam or self-promotion without permission
- Use appropriate channels for questions
- Report scams and impersonators immediately

### Security Warnings

- **Never share your private keys or seed phrase**
- **GlazeCorp will never DM you first**
- **Verify contract addresses before interacting**
- **Beware of fake tokens and scam sites**

## Contributing

### Report Bugs

Found a bug? Open an issue on GitHub with:
- Description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

### Security Issues

**Do NOT publicly disclose security vulnerabilities.**

Contact the team privately through:
- Discord DM to moderators
- Email (if provided)
- Private GitHub issue

### Improve Documentation

Spotted an error in the docs? PRs welcome! Fork the repo and submit improvements.
